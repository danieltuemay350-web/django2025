
# CLI Todo App

A simple command line todo app written in Python.
Tasks are saved to a local JSON file so they persist after you close the program.

---

## Requirements

- Python 3 installed

Check:
```

python --version

```


---

## Run the app

```

python todo.py

```

(or `python3 todoapp.py` if needed)

---

## Commands

Add a task
```

add Buy milk

```

List tasks
```

list

```

Mark task done
```

done 1

```

Delete task
```

delete 1

```

Exit
```

exit

```
